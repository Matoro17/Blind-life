# 🕶️ Blind Code – Jogo de Tabuleiro

**Blind Code** é um jogo de dedução social onde um jogador vendado tenta decifrar palavras com a ajuda (ou sabotagem) dos demais jogadores. Inspirado em jogos como *When I Dream*, o jogo promove tensão, cooperação e traição em partidas rápidas e acessíveis.

## 🎯 Objetivo do Jogo

Os jogadores se revezam como o **Viajante** (vendado), tentando adivinhar **Palavras-Código** a partir das dicas dos demais jogadores. Os outros jogadores recebem papéis secretos como:

- 👁️ **Guias** – tentam ajudar o Viajante a acertar.
- 🕵️ **Infiltrados** – tentam levá-lo ao erro.

## 📦 Componentes

- 🃏 90 Cartas de Código  
- 🎭 9 Cartas de Personagem:
  - 1 Viajante  
  - 5 Guias  
  - 3 Infiltrados
- 🟦 1 Venda  
- ⏳ 1 Ampulheta de 2 minutos  
- 🧩 Conjunto de Marcadores em Braille (letras A–Z)  
- 📄 2 Folhas de Referência do Alfabeto Braille  

## 👥 Preparação

1. Escolha aleatoriamente quem será o primeiro **Viajante** e entregue-lhe a venda.
2. Embaralhe e distribua **1 carta de personagem** para cada jogador (exceto o Viajante).
3. Coloque o **baralho de Cartas de Código** no centro e revele a primeira.
4. Inicie o cronômetro de **2 minutos**.

## 🕹️ Como Jogar

Durante os 2 minutos:

- Apenas o **Viajante vendado** pode falar.
- Ele pode dizer:

  - `"Code"` → para ouvir uma dica do próximo jogador (em ordem horária).
  - Uma **palavra** → para tentar adivinhar a **Palavra-Código**.

### Regras Importantes

- Cada **jogador dá apenas uma dica por carta** (uma palavra apenas).
- **Cada tentativa de adivinhação consome a carta**:

  - ✅ Se acertar, a carta vai para a pilha de pontos dos **Guias e do Viajante**.
  - ❌ Se errar, a carta vai para a pilha dos **Infiltrados**.

- A rodada continua até o tempo acabar, revelando uma nova carta após cada tentativa.

## 🧮 Pontuação

Ao final da rodada:

- Cada carta **acertada**:  
  +1 ponto para o **Viajante** e para **cada Guia**.
- Cada carta **errada**:  
  +1 ponto para **cada Infiltrado**.

## 🔁 Fim do Jogo

O jogo termina quando **cada jogador tiver sido o Viajante uma vez**.  
O jogador com mais pontos vence!

## 🧠 Desafio Final: Braille

Durante a partida, jogadores podem acumular **Marcadores em Braille**.

No final:

- Cada jogador tenta **identificar as letras de suas peças**.
- Cada acerto **dobra os pontos** daquela peça.

Use as **folhas de referência** para validar.

## ⚡ Modo Alternativo: "Código Único"

- O Viajante vence a rodada **ao acertar uma única palavra**.
- Ideal para partidas rápidas ou introdutórias.
